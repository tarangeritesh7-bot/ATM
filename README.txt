ATM Simulation System

This is a simple ATM simulation application written in Java.

Features:
- Account management with balance and PIN
- Deposit, withdraw, and check balance operations
- Input validation utilities

To run the application:
1. Compile the Java files: javac src/com/atm/main/ATMApp.java src/com/atm/model/Account.java src/com/atm/service/ATMService.java src/com/atm/util/InputUtil.java
2. Run the main class: java -cp src com.atm.main.ATMApp
